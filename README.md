# Exchangio 💱

Exchangio is a lightweight Python library that provides simple built-in functions to convert **US Dollars (USD)** to **Indian Rupees (INR)** and **Indian Rupees (INR)** to **US Dollars (USD)**.

Perfect for beginners and Python developers who need quick currency conversions with minimal code.

---

## ✨ Features

- Convert USD to INR
- Convert INR to USD
- Simple built-in functions
- Beginner-friendly API
- Easy installation using pip

---

## 📦 Installation

Install the package using pip:

```bash
pip install exchangio
```

---

## 🚀 Quick Start

```python
import exchangio as ex

# Convert USD to INR
print(ex.ToRupee(10))

# Convert INR to USD
print(ex.ToDol(1000))
```

---

## 📖 Functions

### Convert USD to INR

```python
import exchangio as ex

rupees = ex.ToRupee(25)
print(rupees)
```

---

### Convert INR to USD

```python
import exchangio as ex

dollars = ex.ToDol(5000)
print(dollars)
```
